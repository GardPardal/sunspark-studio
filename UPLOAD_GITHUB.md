# 📤 UPLOAD PARA GITHUB - MANUAL

## Opção 1: GitHub Web Interface (MAIS FÁCIL)

### Passo 1: Ir para seu repositório

```
https://github.com/GardPardal/sunspark-studio
```

### Passo 2: Clicar em "Add file"

```
Botão verde "Code" no topo
└─ Dropdown: "Add file"
└─ Selecionar: "Upload files"
```

### Passo 3: Arrastar e Soltar

```
Copiar estes arquivos da sua pasta:

NOVO/MODIFICADO:
- COMECE_AGORA.md
- CPANEL_OPCOES.md
- README_CPANEL.md
- QUICK_SETUP.md
- START.md
- GODADDY_DEPLOYMENT.md (atualizado)
- MAPA_DE_NAVEGACAO.md (atualizado)
- INDICE_COMPLETO.md (atualizado)

E também:
- src/lib/editable-content.ts
- src/routes/_authenticated/admin.tsx
- build.bat
- build.sh
```

### Passo 4: Commit

```
Message: "Atualizar documentação e painel admin"
Click "Commit changes"
```

**PRONTO!** ✅

## Opção 2: GitHub Desktop (RECOMENDADO)

### Passo 1: Instalar

```
https://desktop.github.com
```

### Passo 2: Clonar seu repositório

```
File → Clone Repository
Selecione: GardPardal/sunspark-studio
Click Clone
```

### Passo 3: Copiar arquivos

```
Cole os arquivos novos em:
C:\Users\organ\Documents\GitHub\sunspark-studio
```

### Passo 4: Commit

```
GitHub Desktop vai detectar mudanças
Click checkbox para selecionar
Preencha "Summary": "Atualizar documentação"
Click "Commit to main"
Click "Push origin"
```

**PRONTO!** ✅

---

## Opção 3: Via VS Code (JÁ TEM)

### Passo 1: Abrir Source Control

```
Ctrl+Shift+G
```

### Passo 2: Inicializar repo (se necessário)

```
"Initialize Repository"
```

### Passo 3: Adicionar Origin

```
Terminal: Ctrl+`
Comando:
git remote add origin https://github.com/GardPardal/sunspark-studio.git
```

### Passo 4: Commit

```
Na aba Source Control:
- Clicar ícone + para "Stage all"
- Digitar message: "Atualizar documentação"
- Ctrl+Enter para commit
- Click "Sync Changes" para push
```

**PRONTO!** ✅
