# 🚀 COPIAR E COLAR - PRONTO!

## ✅ SOLUÇÃO SIMPLES: Vercel (Rápido + Funciona)

### Passo 1: Copiar a Pasta do Projeto

```
Copie esta pasta inteira:
c:\Users\organ\Downloads\sunspark-studio-main\sunspark-studio-main
```

### Passo 2: Fazer Login Vercel

```
1. Abra: https://vercel.com
2. Click "Sign up"
3. Use Gmail ou GitHub
4. Confirme
```

### Passo 3: Conectar Repositório

```
1. Click "Add New Project"
2. Click "Import Git Repository"
3. Cole a URL do seu Git:
   https://github.com/SEU_USUARIO/sunspark-studio
```

### Passo 4: Variáveis de Ambiente

```
Se Vercel pedir "Environment Variables":

Nome                          | Valor
------------------------------|--------------------
SUPABASE_URL                  | [da sua conta Supabase]
SUPABASE_SERVICE_ROLE_KEY     | [da sua conta Supabase]
SUPABASE_PROJECT_ID           | [da sua conta Supabase]
NODE_ENV                      | production
```

### Passo 5: Deploy

```
Click "Deploy"
Aguarde 5-10 minutos
Pronto! Seu site está vivo! ✅
```
